## 개발 계기 

게임개발 스터디에서 개별적으로 게임 개발에 대해 공부하던 도중 유니티 무료 에셋(Asset)을 활용하여 2D 플랫포머 어드벤처 게임을 개발하기로 결정

<br>

## 개발 기획
* 스테이지마다 배경과 타일을 조금씩 다르게 구성하여 총 3개 스테이지를 기획
* 플레이어는 젬을 획득하여 다음스테이지로 넘어갈 수 있다.
